# 学习日志模板

写入路径：`学习日志/YYYY-MM-DD.md`

```markdown
# Session: YYYY-MM-DD

## 今日概览
- **主题**: [今天学了哪些知识点/做了哪些题]
- **时长**: [大约学了多久]
- **模式**: [错题解析 / 概念学习 / 测试 / 复习]

## 学到了什么
- [具体掌握的知识点和关键理解]

## 卡壳与挣扎
- [哪些概念没搞懂？哪些题目做错了？错在哪一步？]

## 今日已掌握（含信心等级）
- [知识点A] - 信心：高/中高/中
- [知识点B] - 信心：高/中高/中

## 下次需要复习
- [具体的知识点或题型]

## 教练评语
- [教练对今天表现的简短点评和建议]
```

## /progress 推荐输入格式

```text
/progress
1. 今天学了什么：
2. 卡住了什么：
3. 做了什么题 / 测试：
4. 状态怎么样：
5. 学了多久：
```

用户也可以自由输入；如果信息不全，教练自行归纳补全，但应围绕以上 5 个字段归档。
