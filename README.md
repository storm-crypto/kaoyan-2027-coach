# Kaoyan 2027 Coach (考研2027 北大软微全能教练)

本项目是一个专门为 **2027年北京大学软件与微电子学院（电子信息专业）四战考生** 量身定制的 AI 智能体技能 (Agent Skill)。

## 🎯 核心使命

针对三战成绩 (政治57，数学一107，英语一77，408 80) 到目标分数 (424分) 的巨大红利空间，本 Skill 提供极度严苛、数据驱动、直击底层的辅导。

核心突破口在于：
1. **数学一 (+28分)**：建立“条件-隐含信息映射”的直觉反射。
2. **408 (+49分)**：彻底打破知识孤岛，重构概念网络。

## ✨ 核心特性

- 🧠 **跨 Agent 记忆系统（Learner Profile）**：
  放弃依赖大模型自身脆弱的上下文记忆。本 Skill 强制要求将复习进度、错题频率、章节掌握度**写入用户本地的 Obsidian Markdown 文件中**。这意味着无论用户切换到 Claude Code、Cursor 还是其他 AI 工具，只要读取本地目录，AI 就能在1秒内“想起”用户是谁、弱点在哪。
- 📐 **数学七维深度解析法**：
  输入错题截图，教练不会直接给答案。而是从“考情分析 → 看到条件大脑该如何反射 → 第一眼直觉是什么 → 避坑指南”等 7 个维度进行思维解剖。
- 💻 **408 三步重构法**：
  直击 80 分以下的“概念混淆”痛点。死磕每个选项，回溯底层原理，提供通俗类比与记忆口诀。
- ⚔️ **残酷抽考与归纳模块**：
  提供 `/test`（随堂高难度概念辨析）和 `/summarize`（章节高频考点梳理），确保真正学懂。
- 📈 **数据驱动的复盘机制**：
  统一沉淀到本地 Obsidian 的"学习者档案 + 学习日志 + 复盘报告"体系中，支持通过 `/weekly_review` 和 `/monthly_review` 生成宏观视角的备考周报/月报。

## 🛠️ 文件结构

*   `SKILL.md`：核心的人设说明、指令逻辑与工作流模板。
*   `README.md`：本项目的介绍文档。

## 🚀 快速上手 (指令列表)

请将此 Skill 加载到支持相关 Agent 框架的客户端中。在与 Agent 的对话中即可使用以下命令：

高频核心命令：
*   `/wrong_math [题目]` - 触发七维数学解析。
*   `/wrong_408 [题目]` - 触发 408 三步解析重构。
*   `/progress [今日心得]` - 记录进度并更新本地学习日志与学习者档案。
*   `/load` - 快速恢复学习状态。

辅助命令：
*   `/explain [概念]` - 请求用大白话解释复杂术语。
*   `/test [章节]` - 要求教练出题抽考概念辨析。
*   `/summarize [章节]` - 梳理本章易混淆知识网络。
*   `/weekly_review` / `/monthly_review` - 生成复盘报告。
*   `/profile` - 读取并更新本地的学习者短板档案。
*   `/anki [知识点]` - 生成可导入 Anki 的问答卡片。
*   `/essay_review [作文]` - 从四个维度批阅考研英语/政治作文。
*   `/score_analysis` - 查看各科差距与当前阶段建议。
