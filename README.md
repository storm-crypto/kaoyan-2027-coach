# Kaoyan 2027 Coach (考研2027 北大软微全能教练)

本项目是一个专门为 **2027年北京大学软件与微电子学院（电子信息专业）四战考生** 量身定制的 AI 智能体技能 (Agent Skill)。

## 核心使命

专为北大软微（电子信息）考研设计，针对用户当前成绩到目标分数的差距空间，提供数据驱动、直击底层的辅导。具体分数和目标以 `我的学习者档案.md` 为准，首次使用通过 `/init` 初始化。

核心突破口在于：
1. **数学一**：建立"条件-隐含信息映射"的直觉反射。
2. **408**：彻底打破知识孤岛，重构概念网络。
3. **政治**：理解底层红线逻辑，突破选择题易混淆陷阱。
4. **英语一**：强化阅读逻辑链分析，保住优势科。

## 核心特性

- **跨 Agent 记忆系统（Learner Profile）**：
  所有复习进度、错题频率、章节掌握度写入用户本地 Obsidian Markdown 文件中。无论切换到哪个 AI 工具，只要读取本地目录，即可在 1 秒内恢复学习上下文。
- **全科错题解析**：
  - 数学七维深度解析法（`/wrong_math`）
  - 408 结构化解析法（`/wrong_408`）
  - 政治红线逻辑解析（`/wrong_politics`）
  - 英语阅读逻辑链分析（`/wrong_english`）
- **主动规划能力**：
  `/plan_week`（周计划）和 `/plan_today`（今日清单），不再只是被动答疑。
- **目标动态调整**：
  `/recalibrate` 输入模考成绩，自动校准各科策略和提分优先级。
- **限时训练模式**：
  `/mock` 模拟考场压力，出题 → 限时 → 批改 → 趋势对比。
- **残酷抽考与归纳模块**：
  `/test`（高难度概念辨析）和 `/summarize`（章节高频考点梳理）。
- **数据驱动的复盘机制**：
  学习者档案 + 学习日志 + 复盘报告体系，支持 `/weekly_review` 和 `/monthly_review`。

## 文件结构

```
kaoyan-2027-coach/
├── SKILL.md                    ← 核心人设、指令逻辑与工作流
├── README.md                   ← 本文档
└── templates/                  ← 按需加载的模板文件（减少主文件 token 消耗）
    ├── 错题追踪卡模板.md        ← 错题卡格式、tags 规则、推荐输入模板
    ├── 学习日志模板.md          ← Session 日志格式、/progress 输入格式
    └── 学习者档案与知识地图模板.md ← 学习者档案结构、全科知识地图（含二级考点）
```

## 快速上手 (指令列表)

**错题解析（核心）：**
* `/wrong_math [题目]` - 触发七维数学解析
* `/wrong_408 [题目]` - 触发 408 结构化解析
* `/wrong_politics [题目]` - 触发政治红线逻辑解析
* `/wrong_english [题目]` - 触发英语阅读逻辑链分析

**规划与训练：**
* `/plan_week` - 生成本周学习计划
* `/plan_today [可用时长]` - 生成今日学习清单
* `/mock [科目] [题量] [时长]` - 限时训练模式
* `/recalibrate [模考成绩]` - 输入模考成绩，校准策略

**进度管理：**
* `/progress [今日心得]` - 记录进度并更新本地学习日志与学习者档案
* `/load` - 快速恢复学习状态
* `/init` - 首次使用时初始化学习者档案与目录结构
* `/weekly_review` / `/monthly_review` - 生成复盘报告
* `/profile` - 读取并更新学习者短板档案
* `/archive` - 归档当前会话上下文到 Obsidian

**辅助工具：**
* `/explain [概念]` - 用大白话解释复杂术语
* `/test [章节]` - 抽考概念辨析
* `/summarize [章节]` - 梳理本章知识网络
* `/串联 [知识点]` - 408 跨学科知识网络
* `/找茬 [推导/代码]` - 像阅卷老师一样挑错
* `/anki [知识点]` - 生成 Anki 问答卡片
* `/essay_review [作文]` - 批阅考研英语/政治作文
* `/score_analysis` - 各科差距与阶段建议
